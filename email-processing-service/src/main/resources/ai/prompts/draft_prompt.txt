- EMAIL_JSON
- INTENT_JSON
- TOOL_OUTPUT_JSON

Output JSON schema (strict):
{
  "draft": "string",
  "confidence": 0.0
}

- confidence must be between 0.0 and 1.0.
- Output JSON only.
